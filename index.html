A study of economic and demographic relationships in relation to Starbucks locations

This is the project page for the Backrow Ballers CS 4354 class project.

Our Datasets

  • business Starbucks Locations This dataset includes a record for every Starbucks or subsidiary store location currently in operation as of February 2017. web
  • language Zip Code Income Data This study provides detailed tabulations of individual income tax return data at the state and ZIP code level. web
  • perm_identity Diversity Dataset This dataset includes diversity metrics for each county web

Objectives

  • Scripts for auto setup of database (if it does not exist)
  • Appropriate tables and relationships to model the data
  • Scripts that analyze and display information about the data in a WebUI
  • Report that reviews any interesting findings using the WebUI

Insights

    There are over 10,000 Starbucks coffee shops across the nation and using this dataset in conjunction with the diversity and income dataset we are able to address these topics:

  • How are the locations of Starbucks stores related to the average amount of income per person?
  • How do Starbucks locations relate to the diversity of a region?
  • Map of Starbucks locations in the US.
  • Areas that have a higher or lower density than expected (as compared to trends).
  • Explore relative distances between Starbucks locations.
